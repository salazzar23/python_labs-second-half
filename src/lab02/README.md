### Реализованный класс коллекции: PatientCollection

Коллекция для хранения и управления объектами `Patient` из ЛР-1. Реализует контейнер для группы пациентов с возможностью добавления, удаления, поиска, сортировки и фильтрации.

---

### Краткое описание класса

`PatientCollection` — это контейнерный класс для управления коллекцией пациентов. Внутри хранит список объектов `Patient`.

**Основные возможности:**

| Категория | Возможности |
|-----------|-------------|
| Хранение | список объектов Patient |
| Управление | добавление, удаление, доступ по индексу |
| Поиск | по ID, имени, диагнозу, специализации врача |
| Сортировка | по возрасту, имени, дате последнего приёма |
| Фильтрация | получение подколлекций (пожилые, срочная помощь, возрастной диапазон) |
| Итерация | поддержка цикла for |
| Защита | проверка типа, запрет дубликатов по patient_id |

---

### Методы коллекции

#### Базовые операции

| Метод | Описание |
|-------|----------|
| `add(item)` | добавить пациента |
| `remove(item)` | удалить пациента |
| `remove_at(index)` | удалить по индексу |
| `get_all()` | получить список всех пациентов |

#### Поиск

| Метод | Описание |
|-------|----------|
| `find_by_id(id)` | поиск по уникальному ID пациента |
| `find_by_name(name)` | поиск по имени (частичное совпадение) |
| `find_by_diagnosis(diagnosis)` | поиск по диагнозу |
| `find_by_doctor_specialization(spec)` | поиск по специализации врача |

#### Магические методы

| Метод | Описание |
|-------|----------|
| `__len__` | получение количества пациентов |
| `__iter__` | поддержка итерации в цикле for |
| `__getitem__` | доступ по индексу (включая отрицательные индексы) |
| `__str__` | строковое представление коллекции |

#### Сортировка (возвращают новую коллекцию)

| Метод | Описание |
|-------|----------|
| `sort_by_age()` | сортировка по возрасту |
| `sort_by_name()` | сортировка по имени |
| `sort_by_last_appointment()` | сортировка по дате последнего приёма |
| `sort(key)` | универсальная сортировка |

#### Фильтрация (возвращают новую коллекцию)

| Метод | Описание |
|-------|----------|
| `get_seniors()` | пожилые пациенты (старше 65 лет) |
| `get_urgent_care()` | пациенты, нуждающиеся в срочной помощи |
| `get_by_age_range(min, max)` | пациенты в возрастном диапазоне |
| `filter(predicate)` | универсальная фильтрация |
